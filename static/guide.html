Section 1

💻 Requirements & Setup

1 Open Your Terminal / Command Prompt

Windows: Press Win + R, type cmd, press Enter
Mac: Press Cmd + Space, type Terminal, press Enter
Linux: Press Ctrl + Alt + T

2 Navigate to the Project Folder
$ cd quickdelivery-webapp
3 Install Dependencies
$ pip install fastapi uvicorn
Collecting fastapi...
Successfully installed fastapi-0.115.0 uvicorn-0.30.0
4 Start the Web Server
$ uvicorn main:app --reload
INFO: Uvicorn running on http://127.0.0.1:8000
INFO: Application startup complete.
✅ Tip
Leave this terminal window open while using the website. Press Ctrl + C to stop the server.
5 Open the Website in Your Browser

Open any browser (Chrome, Firefox, Safari) and go to:

http://localhost:8000

3 Process Orders Step-by-Step

Follow this order workflow:

⏳ Pending → ⚙ Processing → ✓ Completed
  • Start Processing — Click when you begin preparing the order
  • Mark Completed — Click when the order has been delivered
  • Cancel — Cancel an order if needed
  • Delete — Remove the order record permanently
4 Export Orders to CSV

Click the "Export Orders" button on the Admin Dashboard to download all orders as a spreadsheet file. You can open this in Excel or Google Sheets for record-keeping and accounting.

💡 Pro Tips for Admins
• Check the dashboard regularly for new pending orders
• The pending badge on "View Orders" shows how many orders need attention
• Export orders weekly for your records
• Keep stock quantities updated to show "Low Stock" or "Out of Stock" to customers
